FONTLOG
Nirmal, Asha, Elur - NLCI Telugu fonts
======================================

This file provides detailed information on the NLCI Telugu fonts.
This information should be distributed along with the NLCI Telugu fonts and any derivative works.

Basic Font Information
----------------------

Pan Telugu font designed to support all the languages using the Telugu script.
