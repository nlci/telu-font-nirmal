NLCI Telugu fonts
=================

This file describes the source files included with the NLCI Telugu fonts.
This information should be distributed along with the NLCI Telugu fonts and any derivative works.

master.gdl            - Graphite rules
master.sfd            - OpenType rules
other .sfd files      - glyph outline data for each font
nirmal_classes.xml    - class information for the attachment point database
nirmal_unicode.xml    - mapping from legacy to Unicode as used by ttfbuilder
TEL Nirmal 2 Unicode CONV chart.xls - source for nirmal_unicode.xml
